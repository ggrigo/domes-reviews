URL Structure & Usage Instructions

Resort Selection
Use URL parameter to select resort:
?resort=domes-white-coast-milos
?resort=domes-noruz-chania
Default: domes-white-coast-milos
Platform Routing
Use platform parameter to set review destination:
?platform=tripadvisor (default)
?platform=google
?platform=booking
?platform=expedia
Combined Parameters
Full URL examples:
domain.com?resort=domes-white-coast-milos&platform=google
domain.com?resort=domes-noruz-chania&platform=booking
QR Code Generation
Generate QR codes with specific parameters for each use case. Place QR codes in rooms, reception, or send via email with appropriate resort and platform parameters.
Adding New Resorts
To add a new resort, update the resorts-config.json file with: features array, staff members, platform URLs, and resort ID. The system will automatically load the new configuration.
